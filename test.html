Project Requirements

  • Model had to include a has_many, belongs_to and a has_many :through relationship.

  • Included class level ActiveRecord scope methods.
  • Allowed for standard user authentication, including OmniAuth functionality.
  • All forms were required to correctly display validation errors.
  • And of course, it had to be DRY.
